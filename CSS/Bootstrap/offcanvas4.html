Bootstrap version used: 4.2.1

This is a basic demo of Bootstrap 4 off-canvas sidebar, based on the Fellow Tuts Article.

Features:

  • Easier for WordPress
  • Dynamic content overlay
  • Left/right or both sidebars
  • Sidebar opener in the navbar as well

Click to the left arrow yellow button at the top-right corner to the navbar button in the action. Clicking to the hamburger icon will also slide in the respective off-canvas sidebar.
